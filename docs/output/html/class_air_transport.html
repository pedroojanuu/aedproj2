Detailed Description

A class that serves the purpose of having all the information about the network and operations over it centralized.
It accommodates pointers to all Airports, Cities and Airlines.
As described in the documentation's main page, the given network is represented in a Graph, where Airports are the nodes and Flights are the edges.

◆ bfs() [1/2]

void AirTransport::bfs ( Airport *  source)

Implementation of Breadth-First-Search given a source airport. It marks the visited airports as so and stores the distance to the source airport, as well as storing, for each airport it traverses, the last visited airport. This is important to build routes.
Complexity: O(|V|+|E|), where |V| is the number of airports and |E| the number of flights.
