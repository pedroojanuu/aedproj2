Detailed Description

A class to represent an airport by its IATA code, name, city (which includes the country) and coordinates.
Since we represent the provided air traffic network by a directed graph, the Airports are the nodes, which leads to having a vector of departing flights (which are the graph's edges), a booleanvisited to represents if we have traversed the node in a search, as well as an integerdistance and a pointer to the airport where we 'came from', in order to build a route.

static

Hash function that returns an integer based on a given IATA code.
Complexity: the complexity of the function's algorithm is O(n), with n being the length of the given code. However, given the context of the program and the provided air traffic dataset, since all IATA codes have exactly 3 characters and the function is only called when it has previously been checked that the given input has 3 characters, the O(n) 'is' O(3), which is O(1) complexity.
