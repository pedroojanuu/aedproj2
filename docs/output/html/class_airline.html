int Airline::hash ( const string &  code)
static

Hash function that returns an integer based on a given ICAO code.
Complexity: the complexity of the function's algorithm is O(n), with n being the length of the given code. However, given the context of the program and the provided air traffic dataset, since all ICAO codes have exactly 3 characters and the function is only called when it has previously been checked that the given input has 3 characters, the O(n) 'is' O(3), which is O(1) complexity.
